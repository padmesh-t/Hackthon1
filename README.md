# ✈ Traveloop – Travel Planning App

A single-file PHP application refactored into a modular structure.  
Manage trips, build itineraries, track budgets, create packing lists, write notes, and share with the community – all with a beautiful dark UI.

## 🚀 Quick Start

1. **Requirements**  
   - PHP 7.4+ with `pdo_mysql` extension  
   - MySQL 5.7+ / MariaDB  
   - Web server (Apache, Nginx, or built‑in)

2. **Clone / download the project**  
   Place the `traveloop` folder in your web root (e.g., `htdocs/traveloop`).

3. **Configure database**  
   Open `config/database.php` and set your MySQL credentials:
   ```php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'traveloop');
   define('DB_USER', 'root');
   define('DB_PASS', '');
